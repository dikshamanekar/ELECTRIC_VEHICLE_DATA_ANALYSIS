# Electric Vehicle Analysis Dashboard (Tableau)

## Project Overview
This project analyzes Electric Vehicle data using Tableau to identify sales trends, vehicle types, and regional adoption patterns.

## Tools Used
- Tableau
- Excel
- Data Visualization
- Data Analysis

## Dataset
Electric Vehicle dataset containing information about vehicle types, sales trends, and regional distribution.

## Dashboard Insights
- EV adoption trends over time
- Top vehicle manufacturers
- Regional distribution of EVs
- Vehicle type analysis

## Project Structure
Data → Dataset used for analysis
Tableau_Workbook → Tableau dashboard file
Dashboard_Images → Dashboard screenshots
Reports → Project documentation

## How to Use
1. Download the `.twbx` file
2. Open in Tableau Desktop
3. Explore the interactive dashboard
